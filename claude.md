## Status Reports

All daily/weekly status reports must follow this filename pattern:

```
DD-MM-YYYY-daily-status.md
```

Example: `13-04-2026-daily-status.md`

This applies to all roles: Tech Lead, Dev 1, Dev 2, Dev 3, P.O.
