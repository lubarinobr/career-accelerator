## 🏗️ Module 1: AI Engine (The Brain)
This module handles the real-time generation of technical content and linguistic adaptation.

* **Feature 1.1 - Dynamic Question Generator:** Integration with an LLM API to generate multiple-choice questions based on specific certification blueprints (e.g., AWS, Azure, PMP).
* **Feature 1.2 - Socratic Feedback System:** An engine that provides brief, technical explanations in English after every answer. It must use "Simplified English" for beginners and "Technical English" for advanced users.
* **Feature 1.3 - Adaptive Difficulty Logic:** An algorithm that analyzes the user's success rate and adjusts the complexity of the next question set to keep the user in the "Flow State."

---

## 🎮 Module 2: Gamification & Retention (The Hook)
These features are designed to fight the "three-year inertia" by providing immediate dopamine hits.

* **Feature 2.1 - The "Streak" Engine:** A visual counter of consecutive days. It includes a "Streak Freeze" mechanic (purchasable with in-app XP) to prevent total demotivation after a single missed day.
* **Feature 2.2 - AI-Driven Motivational Notifications:** Push notifications that aren't generic. The AI analyzes the time you usually procrastinate and sends a "call to action" in English, varying from encouraging to "tough love" (Drill Sergeant mode).
* **Feature 2.3 - XP & Leveling System:** Users earn XP for correct answers and "Perfect Lessons." Levels should correspond to professional titles (e.g., Intern -> Junior -> Senior -> Certified).

---

## 📱 Module 3: Core Learning UI (The Experience)
The interface must focus on "zero friction." Opening the app to answering the first question should take less than 3 seconds.

* **Feature 3.1 - Micro-Quiz Interface:** A "Tinder-style" or "Duolingo-style" UI where users see one question at a time with clear, large touch targets.
* **Feature 3.2 - Integrated Text-to-Speech (TTS):** A button to play the audio for every question and explanation, helping with English listening and pronunciation.
* **Feature 3.3 - Focus Mode:** A timer-based session (e.g., 5 or 10 minutes) that locks the user into the quiz to prevent app-switching.
